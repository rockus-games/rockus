1. На рабочем столе создать новый каталог с названием "Линейная классификация"
2. Скачать из сайта https://rockus.su/webinfo/FC/files/files.html файлы № 2, 3, 4
3. Все скачанные файлы перенести в созданный ранее каталог "Линейная классификация"
4. Открыть каталог в VSCode
5. Переименовать файл liner_classification.txt на liner_classification.py
6. Создать виртуальное окружение прописав в терминале команду: python -m venv venv
7. Закрыть и заново запустить терминал для активации виртуального окружения.
8. Установить библиотеку sklearn прописав в терминале команду: pip install scikit-learn==1.1.3
9. Установить библиотеку pandas прописав в терминале команду: pip install pandas
10. Запустить liner_classification.py
11. Если всё работает, то  выложить liner_classification.py в МЭО как практическая работа линейной классификации в 3 модуле.
